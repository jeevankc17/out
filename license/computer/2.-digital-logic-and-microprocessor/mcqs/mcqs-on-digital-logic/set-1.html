1. Hexadecimal can express in one digit what is done by following binary number of binary digits:

  1. One\text{One}One
  2. Four\text{Four}Four
  3. Two\text{Two}Two
  4. Eight\text{Eight}Eight
Show me the answer

Answer: 2. Four\text{Four}Four

Explanation:

  • Hexadecimal (base-16) uses one digit to represent 4 binary digits (bits).
  • This is because 24=162^4 = 1624=16, so one hexadecimal digit can represent 16 unique values, equivalent to 4 binary digits.

2. NAND gates are preferred over others because these have:

  1. Have lower fabrication area\text{Have lower fabrication area}Have lower fabrication area
  2. Can be used to make any gate\text{Can be used to make any gate}Can be used to make any gate
  3. Consume less electronic power\text{Consume less electronic power}Consume less electronic power
  4. Provide maximum density in a chip\text{Provide maximum density in a chip}Provide maximum density in a chip
Show me the answer

Answer: 2. Can be used to make any gate\text{Can be used to make any gate}Can be used to make any gate

Explanation:

  • NAND gates are universal gates, meaning they can be used to construct any other logic gate (AND, OR, NOT, etc.).
  • This flexibility makes them highly preferred in digital circuit design.
  • For example:
    • A NOT gate can be made using a NAND gate by connecting both inputs together.
    • An AND gate can be made by combining two NAND gates.

3. According to De Morgan’s theorem, A+B‾=\overline{A + B} =A+B​=

  1. A‾⋅B‾\overline{A} \cdot \overline{B}A⋅B
  2. A‾+B‾\overline{A} + \overline{B}A+B
  3. A⋅BA \cdot BA⋅B
  4. A+BA + BA+B
Show me the answer

Answer: 1. A‾⋅B‾\overline{A} \cdot \overline{B}A⋅B

Explanation:

  • De Morgan’s theorem states that the complement of a sum is equal to the product of the complements:
    A+B‾=A‾⋅B‾\overline{A + B} = \overline{A} \cdot \overline{B}A+B​=A⋅B
  • This is one of the fundamental laws in Boolean algebra and is widely used in simplifying logic expressions.

4. According to De Morgan’s theorem, A⋅B‾=\overline{A \cdot B} =A⋅B=

  1. A‾+B‾\overline{A} + \overline{B}A+B
  2. A‾⋅B‾\overline{A} \cdot \overline{B}A⋅B
  3. A+BA + BA+B
  4. A⋅BA \cdot BA⋅B
Show me the answer

Answer: 1. A‾+B‾\overline{A} + \overline{B}A+B

Explanation:

  • De Morgan’s theorem states that the complement of a product is equal to the sum of the complements:
    A⋅B‾=A‾+B‾\overline{A \cdot B} = \overline{A} + \overline{B}A⋅B=A+B
  • This is another fundamental law in Boolean algebra and is used to simplify complex logic expressions.
